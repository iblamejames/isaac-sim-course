## 📌 О курсе

Курс охватывает полный цикл работы с **NVIDIA Isaac Sim** — от установки и первого запуска до создания сцен с роботами и написания собственных скриптов управления. Каждое занятие включает теорию, демонстрацию через анимированные примеры и практическое задание.

После прохождения курса вы сможете самостоятельно строить физические симуляции, добавлять промышленных роботов и управлять ими через Python код — это основа для дальнейшей работы с обучением роботов, сенсорами и интеграцией с ROS.

### Для кого этот курс?

- Разработчики робототехнических систем
- Исследователи в области RL и имитационного обучения
- Студенты, изучающие робототехнику и AI
- Все, кто хочет освоить физическую симуляцию на базе USD/PhysX

## ⚙️ Системные требования

| Компонент | Минимум | Рекомендуется |
|---|---|---|
| Видеокарта | NVIDIA RTX 3080 (10 ГБ VRAM) | NVIDIA RTX 4080 и выше |
| Оперативная память | 32 ГБ | 64 ГБ |
| Место на диске | 50 ГБ SSD | 100 ГБ NVMe SSD |
| ОС | Windows 10/11 (64-bit) | Windows 11 |

> **Важно:** карты AMD, Intel и серия NVIDIA GTX не поддерживаются. Требуется архитектура RTX с аппаратной трассировкой лучей.

## 📚 Программа курса

| # | Занятие | Темы | Практическое задание |
|---|---------|------|----------------------|
| 1 | [**Введение в Isaac Sim**](./lesson1.md) | Установка, интерфейс (Viewport, Stage, Property, Content, Toolbar), навигация, первый запуск симуляции | Найти зоны интерфейса, создать первую сцену с объектом |
| 2 | [**Объекты и физика**](./lesson2.md) | Prim, Physics Scene, Ground Plane, Rigid Body, Collision, трансформация (W/E/R), Physics Material | Сцена с падающими объектами, эксперимент с упругостью |
| 3 | [**Роботы и первые шаги с кодом**](./lesson3.md) | Franka Panda, структура робота (звенья и суставы), сборка собственного робота, Script Editor, Python API | Собрать двухколёсного робота, воспроизвести сцену из урока 2 кодом |
| 4 | [**Управление роботом через код**](./lesson4.md) | Dynamic Control API, задание позиций суставов, чтение текущего состояния, управление несколькими суставами | Задать позу роботу через скрипт, прочитать и сравнить состояние суставов |

## 🚀 Быстрый старт

1. Убедитесь что ваш ПК соответствует системным требованиям
2. Пройдите урок 1 — он содержит инструкцию по установке
3. Проходите уроки последовательно: каждый опирается на предыдущий
4. Выполняйте практические задания — без практики теория не закрепляется
